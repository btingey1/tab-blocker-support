Tab Blocker Test

Use this page to verify that Tab Blocker
is blocking unwanted new tabs.

Test: Open a New Tab

Tap the button below. It will attempt to open a new tab via JavaScript. If Tab Blocker is enabled for this site, the new tab will be blocked.

Test: Auto Pop-up

This simulates an ad-style pop-up that opens automatically. Tap the button to trigger it after a 2-second delay.

How to Test

1. Open the Tab Blocker app and follow the setup steps
2. In Safari, tap the extensions icon (puzzle piece) on this page
3. Tap Tab Blocker and enable it for this site
4. Reload this page, then tap the test buttons above
5. If blocked, the new tab will not open and a green confirmation appears

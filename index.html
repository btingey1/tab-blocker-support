How It Works

Tab Blocker is a Safari extension that prevents websites from opening unwanted new tabs. Enable it per-site with a single tap.

Setup

  • Open Settings → Safari → Extensions
  • Enable Tab Blocker
  • Set permissions to "All Websites" or choose per-site
  • In Safari, tap the extension icon to activate on any site

Troubleshooting

  • Extension not working? Reload the page after enabling. Tab Blocker activates on page load.
  • Site broken? Tap the extension icon to disable Tab Blocker for that site.
  • Ads still opening tabs? Make sure permissions are set to "Allow" for the site, then reload.
